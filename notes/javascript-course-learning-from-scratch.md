## console.log

printing out anything is done via `console.log`

to print your name you write

```js
console.log(`my name is philip`);
```

click on 'run' button and you should see

```js
// my name is philip
```

to print your name as a variable you can write

```js
console.log(`my name is philip`);
let myName = `arooj`;
console.log(`my name is ${myName}`);
myName = `paula`
console.log(`my name is ${myName}`);
myName = `hacker`
console.log(`my name is ${myName}`);
/*
my name is philip
my name is arooj
my name is paula
my name is hacker
*/
```



## counting to 100

to print all the numbers 1 to 5 we can write


```js
for (let i=0; i<=5; i++) {
    console.log(`your number is ${i}`);
}
/*
your number is 0
your number is 1
your number is 2
your number is 3
your number is 4
your number is 5
*/
```



## adding numbers

